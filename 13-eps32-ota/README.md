
## ESP-IDF Programming Guide

```sh
idf.py help

idf.py create-project demo
cd demo
idf.py set-target esp32
idf.py menuconfig
edit demo/main.c
idf.py build
idf.py -p /dev/ttyUSB0 flash

idf.py monitor
```
